# Caminho do Desenvolvedor

Bem-vindo. Este repositório é o seu mapa para entrar na área de tecnologia com um rumo claro. Não é mais uma lista interminável de links, nem um daqueles roadmaps genéricos que jogam duzentos tópicos na sua tela e dizem "estude isso". Aqui você encontra **guias completos de estudo** — cada um escrito como se um professor estivesse ao seu lado, explicando o que estudar, como estudar e quando estudar.

## Para que serve

Este projeto existe para responder a três perguntas que todo iniciante (e mesmo profissionais em transição) se fazem:

1. **O que** eu preciso saber para dominar uma área?
2. **Como** eu estudo isso de forma eficiente e sem esquecer?
3. **Quando** eu vou estar pronto para o mercado?

Cada guia entrega as três respostas num só lugar, organizadas numa sequência lógica e com uma metodologia de aprendizado contínuo.

## Objetivo

Levar você **do zero até o nível de referência** — aquele profissional que outros consultam, que decide arquiteturas, que resolve problemas complexos. Não é sobre decorar sintaxe ou passar em entrevistas. É sobre construir uma base sólida que sustenta uma carreira inteira.

## Público

- Pessoas começando do absoluto zero, sem experiência em programação
- Profissionais de outras áreas fazendo transição para TI
- Desenvolvedores iniciantes que querem organizar os estudos e preencher lacunas
- Qualquer pessoa que prefere um caminho estruturado a um monte de conteúdos soltos

## Metas

- **Clareza**: cada guia explica o *porquê* de cada tópico, não apenas o *o quê*
- **Completude**: cobrir desde os fundamentos até tópicos avançados, sem pular etapas
- **Prática**: projetos reais em cada fase, porque teoria sem mão no código não fixa
- **Sustentabilidade**: uma metodologia de ciclo contínuo que impede a sensação de "nunca sei o suficiente"

## Como usar este repositório

1. **Escolha uma área** — comece por aquela que mais combina com seus objetivos
2. **Leia o guia do começo ao fim** — ele foi escrito na ordem certa; pular etapas é o caminho mais rápido para a ansiedade
3. **Estude na prática** — cada fase tem projetos para você construir. Não adianta só ler
4. **Aplique o ciclo** — a metodologia de 6 meses garante que você revisa, aprofunda e nunca fica parado
